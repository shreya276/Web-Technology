ğŸ§± Core HTML Tags Covered

1. Headings (<h1> to <h6>)

  • Define the hierarchy and structure of content.
  • <h1> represents the main heading, while <h6> denotes the least important heading.

2. Paragraph (<p>)

  • Used for writing blocks of text.
  • Browsers automatically add spacing before and after paragraphs.

3. Line Break (<br>)

  • Inserts a line break within text.
  • Useful for poems or addresses where line breaks are significant.

4. Horizontal Rule (<hr>)

  • Creates a thematic break or horizontal line.
  • Often used to separate content sections.

5. Bold (<b>) and Strong (<strong>)

  • <b> makes text bold without semantic importance.
  • <strong> indicates that the text is of strong importance.

6. Italic (<i>) and Emphasis (<em>)

  • <i> italicizes text without semantic importance.
  • <em> emphasizes text, indicating stress or importance.

7. Underline (<u>)

  • Underlines text.
  • Typically used for stylistic purposes.

8. Superscript (<sup>) and Subscript (<sub>)

  • <sup> raises text above the baseline (e.g., exponents).
  • <sub> lowers text below the baseline (e.g., chemical formulas).

ğŸ“ Semantic Importance

The video emphasizes the significance of using semantic tags to enhance accessibility and SEO:

  • Semantic Tags: Tags like <strong> and <em> convey meaning, aiding screen readers and search engines.
  • Non-Semantic Tags: Tags like <b> and <i> are purely stylistic and don't convey additional meaning.
